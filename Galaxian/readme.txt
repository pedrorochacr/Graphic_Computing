Instruções para compilação:
-Basta realizar o build do arquivo no codeBlocks e iniciar o arquivo.exe dentro da pasta bin do projeto. Isso tudo é realizado com as devidas 
instalações do OpenGl no computador.O projeto foi executado no Windows.

-Teclas a e d do teclado movimentam o jogador para esquerda e direita, respectivamente.
-Tecla espaço atira.
-Tecla r reinicia o jogo.
-Tecla p pausa o jogo
-Esc sai da aplicação
